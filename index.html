üîπ Step 1: HTML Structure

Create a file called index.html and add the following code:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Responsive Navbar</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <nav class="navbar">
    <div class="logo">CodeWithDeepesh</div>
    <ul class="nav-links">
      <li><a href="#">Home</a></li>
      <li><a href="#">Tutorials</a></li>
      <li><a href="#">Projects</a></li>
      <li><a href="#">Contact</a></li>
    </ul>
  </nav>

</body>
</html>

üîπ Step 2: CSS Styling

Create a file called style.css and style the navbar using Flexbox:

body {
  margin: 0;
  font-family: Arial, sans-serif;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background: #333;
  padding: 1rem 2rem;
  color: white;
}

.logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
}

.nav-links a:hover {
  text-decoration: underline;
}

üîπ Step 3: Make It Responsive

To hide the menu on smaller screens (below 600px), add this media query:

@media (max-width: 600px) {
  .nav-links {
    display: none;
  }
}

‚úîÔ∏è Now, when you shrink the screen, the nav items disappear ‚Äî this sets up the base for a mobile hamburger menu (which you can add later with JavaScript).
